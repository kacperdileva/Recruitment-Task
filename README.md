# RecruitmentTask
Simple app with API connected to Retrofit

 
 # Tech Stack
 
 * Kotlin
 * MVVM 
 * ViewBinding
 * Dagger + Hilt
 * Retrofit  + Gson + Callback library
 * Room
 * Flows ( Channels ) 
 * Search View
